Basic Web Structure

The world wide web basically consists of four major bricks:

1. Browser

2. Internet

3. HTTP

4. Server

If you enter a URL in your browser, your pc sends an HTTP-Request to a server, which is used to find
the specific HTML-document on the server, which is then send back to your PC. Why an HTML-
document is needed will be explained in the next chapter.

HTML Documents

HTML-Documents are used to describe a Web-Page. These documents
need to follow a specific structure , so that everyone is able to understand the code. HTML
-Documents need to contain the following points:

1. Document-Type

2. HTML-Tag

3. Head

4. Body

Tags and Elements

HTML-Documents are containing Tags and Elements. These are used to "explain" the browser what it sould show on the screen.

<TAG> Normal Tags </TAG>

A Normal Tag contains a opening and closing tag, which are surounding the context. For example:
<b> opening and closing </b> is showen as opening and closing

<TAG> Void Tags

A Void Tag does not contain a closing tag! For example <br>

Inline and Boxelements

There are Inlineelements , which are limited by a line
and
Boxelements
